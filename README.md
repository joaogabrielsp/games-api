<h1>API de Jogos</h1>

<h2>📚 Sobre o projeto</h2>

Esta API RESTful foi desenvolvida em Java com Spring Boot e segue o padrão de camadas Controller, Service, Repository e DTO.
Ela permite a manipulação e consulta de uma base de dados de jogos, fornecendo endpoints para recuperar e modificar listas de jogos.

<h2>Tecnologias utilizadas</h2>

- Java
- Spring Boot
- Maven
- PostgreSQL (Banco de dados principal)
- H2 Database (Banco de dados em memória para testes)
- Postman (Testes de API)
